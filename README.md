# Methodology: AI ROI Calculation

## Data Collection (2022-2025)

200 AI projects tracked via:
- Client CRM/ERP data (anonymized exports)
- Quarterly interviews (structured questionnaire)
- Financial audits (invoices, timesheets)

## Variables Measured

### Input Variables
- Industry sector
- Company size (employees, revenue)
- AI use case type
- Budget allocated
- Team size involved

### Output Variables  
- Time saved (hours/week)
- Revenue increase (€)
- Costs avoided (€)
- Implementation time (weeks)
- Adoption rate (% team using AI daily)

## Quality Controls

1. **Outlier Detection**: Removed 12 projects (6%) with ROI > 1000% (unverifiable)
2. **Survivorship Bias**: Included 23 failed projects (11.5%)
3. **Self-Reporting Bias**: Cross-checked 30% sample with financial data

## Confidence Levels

- High confidence (80%+): 50+ projects in segment
- Medium confidence (60-80%): 20-50 projects
- Low confidence (<60%): <20 projects (marked in dataset)
